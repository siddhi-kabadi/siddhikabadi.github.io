01

The Problem

Climate change and inefficient irrigation continue to strain water resources globally. Small community farms — often operating on shoestring budgets — face severe challenges maintaining their crops. While smart irrigation solutions exist, they are typically cost-prohibitive for nonprofits and community organizations.

"Data-driven farming made simple — empowering urban farming initiatives."

Deployed at Veggielution, a 6-acre urban farm in San Jose promoting youth education and fresh produce distribution to underserved communities. Their staff provided continuous field feedback to refine our sensor system for real farming conditions.

System Architecture

Solar + Battery
3D-Printed PETG
→ LoRa →
📡 Base Station Adafruit Feather M0
LoRa Radio Module
Wi-Fi Forwarding
→ WPA2 →
📊 Web Dashboard Next.js + React
Mapbox + Chartus
MongoDB + Express

Sensing units transmit soil moisture readings over LoRa to the base station, which forwards encrypted data via WPA2 to backend servers. The dashboard displays real-time readings and triggers automated valve activation when moisture drops below set thresholds.

04

Hardware Features

Soil Moisture Probes4 probes per sensing unit for cross-validation, ensuring accuracy across varied soil conditions
Power SystemRechargeable batteries charged via integrated solar panels — continuous off-grid operation
Enclosure3D-printed PETG casing with UV resistance, venting features, and custom gaskets for all weather
LoRa ModulesLong-range, low-bandwidth communication across multiple acres while minimizing power draw
EACP ProtocolCustom Energy-Aware Communication Protocol — dynamically adjusts transmission frequency based on battery charge for maximum longevity
SecurityWPA2 encryption for all data transmission to backend servers, protecting against packet loss and tampering
